### ORM的发展历程
java - SSH、SSM
SSH:spring+struts+Hibernate
SSM:spring+springMVC+Mybatis
Hibernate、Mybatis => ORM框架

当前主流：微服务化、前后分离

- 微服务是各个应用变小、变轻
- 前后分离则替代了mvc

SSH、SSM配置文件非常复杂，搭建起来麻烦(配置原因、版本冲突、其他原因),开发和维护起来也非常麻烦；
